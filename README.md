# 🔥 तेरा नाम क्या बोलता है? (Tera Naam Kya Bolta Hai?)

> AI bata dega tu kaisa insaan hai 😂

A viral, share-worthy name analyzer for Indians. Built in React, zero API needed, fully client-side.

## Features
- 🎭 20 unique personality archetypes (10 boys + 10 girls)
- 💕 Dating life predictions
- 🚩 Red flag meter
- 🎬 Bollywood character matches
- 📸 Downloadable result cards
- 📲 WhatsApp share built-in
- ⚡ 100% free, no signup, no API
